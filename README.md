#------------ Running the app ------------
Install Meteor with:
curl https://install.meteor.com | sh

Run with:
meteor
#------------ Compiling mini-games ------------
Run the file:
./private/games/kickjs 

It should give the output:
Compiling punchbags...Done
Compiling refugees...Done

Go to this URL to see the game:
http://localhost:3000/games/
